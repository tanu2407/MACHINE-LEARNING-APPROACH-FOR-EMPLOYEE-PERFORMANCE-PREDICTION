About This Project

This project uses advanced machine learning algorithms to predict employee productivity based on various factors such as targeted productivity, department, team, over time, idle time, number of style changes, SMV, incentives, and worker details.

It is built using Python, Flask, and Scikit-learn, with models including Linear Regression, Random Forest, and XGBoost. The system compares model performances and selects the best one for predictions.

This application provides an easy-to-use interface for businesses to assess employee performance and make data-driven decisions.
